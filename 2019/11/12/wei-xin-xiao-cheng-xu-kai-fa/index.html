语法

使用动态数据

使用插值表达式在页面渲染数据，如下

//1.index.js  配置
Page({
  data:{
    msg:"wechat"
  }
})
//2.index.wxml
<view class="container">
    //插值表达式  可以使用变量,使用javascript表达式
     {{msg}}
</view>

通过{{msg}}可以获取到index.js里data里属性msg的属性值，并将其渲染到界面

列表渲染 wx:for

wx:for类似于一个循环，将要循环的数组内需要的对象属性渲染到相应位置，如下

//index.js
Page({
  data: {
    friends: [
      { id: 1000, title: "h6" },
      { id: 1001, title: "html" },
      { id: 1002, title: "css" }
    ]
  }
})
//index.wmxl
<view class="container" wx:for="{{friends}}" wx:key="{{id}}">
  {{item.title}}
</view>

使用wx:for="{{friends}}"对frinends这个data里的friends数组进行循环，并通过item.属性名将相应的属性值的循环结果呈现在界面

绑定事件

使用bindtap在相应的标签内绑定相应的事件

举个例子：通过点击页面中的字来改变它的内容

//index.js
Page({
  data: {
    msg:"hello world"
  }
})
//index.wxml
<view bindtap="handleClick">
  {{msg}}
</view>

页面的数据应该是hello world，我们现在在数据的view标签上绑定了一个事件handleClick,接下来我们在js内设置这个事件的内容：

//index.js
Page({
  data: {
    msg:"hello world"
  },

  handleClick(){
  //this.setData() 设置data中的值
    this.setData({
        msg:"change"
    })
  }
})

在handleClick事件内我们使用this.setData()设置data里的属性的属性值，将呈现在页面的msg属性改为“change”，此时点击的时候触发点击事件就会改变页面的字为“change”了

自定义属性的设置和获取

自定义属性的设置

使用data-自定义属性名来自定义一个属性，例如：

//1.data-id 设置一个自定义属性id
<view data-id="{{id}}" bindtap="handleClick">
    {{msg}}
</view>

在这个使用data-id自定义属性id,并且将data内的id值传给这个自定义属性，比如在data里设定id值为1001

Page({
    data:{
        msg:"hello world",
        id:1001
    }
})

自定义属性的获取

通过以上例子已经设置好了一个自定义属性并且把已有的id的值传给了这个自定义属性，接下来我们怎么获取这个值呢？我们通过一个点击事件来演示一下这个自定义属性值的获取：

Page({
    data:{
        msg:"hello world",
        id:1001
    },

    handleClick(event){
        console.log(event)
        console.log(event.currentTarget.dataset)
    }
})

通过在点击事件传一个event参数
我们通过打印event可以看到一些已有的属性

在event.currentTarget里可以看到一些现成的数据，其中dataset里就是自己设置的自定义属性的值了

此时可以看到id确实在这里面，通过打印event.currentTarget.dataset.id就可以获得1001了

获取data中的值

通过this.data.属性就可以获得到data 内相应的属性了

//index.js
Page({
  data:{
    msg:"hello world",
    arr:[{id:1,title:"xxx"},{id:2,title:"yyy"},    {id:3,title:"zzz"}],
    id:1001
  },
  handleClick(event){
    console.log(this.data.arr[2].title);        //zzz
  }
})

列表删除

通过以上的学习我们已经使用wx:for="{{arr}}",将数组内的数据循环渲染到了界面，我们这时候需要使用到自定义属性，通过自定义属性传一个当前循环的属性，通过这个属性值和循环的数组内的属性的相应属性值比对，找到相同的排除，保留点击相同的就可以实现点击删除了。

1、首先先循环渲染页面

使用wx:for="{{arr}}"

2、设置一个自定义属性，绑定点击事件

设置自定义属性data-id="{{item.id}}"，将自定义属性的值设置为循环的当前id，这样我们就可以通过事件得到当前点击的id并和原数据内的数据比对了

以上两步完成代码如下

//index.js
Page({
  data:{
    msg:"hello world",
    arr:[
        {id:1,title:"xxx"},
        {id:2,title:"yyy"},
        {id:3,title:"zzz"}],
    id:1001
      }
  }
})
//index.wxml
<view wx:for="{{arr}}" wx:key="{{item.idid}}"  data-id="{{item.id}}"
bindtap="handleClick">
    {{item.title}}
</view>

此时页面循环渲染结束

3、通过点击事件和自定义属性删除相应属性

因为一开始我们自定义属性绑定的就是相应的id属性，所以我们获取到点击的自定义属性的id值，和原有的数组内的id值比对，相同的过滤掉，就可以实现点击过滤效果，代码如下

//index.js
handleClick(event){
    let arr2 = this.data.arr.filter(value=>{
      return value.id != event.currentTarget.dataset.id
    })
    this.setData({
      arr:arr2
    })
  }

event.currentTarget.dataset.id获得的就是点击的那条对象的id值，通过filter过滤掉

4、最后将过滤后的结果返回给data

上一段代码已经写出来了，通过this.setData将处理好的arr2传回data
