Vue基本语法

插值操作

Mustache

Mustache语法:双大括号

Mustache:胡须/胡子

<div id="app">
        {{message}}
        <h1>{{firstName+' '+lastName}}</h1>
        <h1>{{counter*2}}</h1>
    </div>
    <script src="../js/vue.js"></script>
    <script>
        let app = new Vue({
            el: `#app`, data: {
                message: '你好啊',
                firstName: 'Kobe',
                lastName: 'Bryant',
                counter: 1900
            }
        })
    </script>

v-once指令

只会在第一次显示数据，之后修改并不会改变

<div id="app">
        <h2 v-once>{{message}}</h2>
    </div>
    <script src="../js/vue.js"></script>
    <script>
        let app = new Vue({
            el: `#app`, data: {
                message: '你好啊'
            }
        })
    </script>

v-html

解析字符串，识别a标签，并且能解析href地址

<div id="app">
        <h2 v-html="url"></h2>
    </div>
    <script src="../js/vue.js"></script>
    <script>
        let app = new Vue({
            el: `#app`, data: {
                url:'<a href="https://www.baidu.com">百度一下</a>'
            }
        })
    </script>

v-text

用起来并不灵活，会覆盖后面的内容

<div id="app">
        <h2 v-once>{{message}},李银河</h2>            //你好啊,李银河
        <h2 v-text="message">,李银河</h2>            //你好啊
    </div>
    <script src="../js/vue.js"></script>
    <script>
        let app = new Vue({
            el: `#app`,
            data: {
                message: '你好啊'
            }
        })
    </script>

v-pre

不解析Mustache语法，将内容原封不动输出

<div id="app">
        <h2 v-pre>{{message}},李银河</h2>        //{{message}},李银河
    </div>
    <script src="../js/vue.js"></script>
    <script>
        let app = new Vue({
            el: `#app`,
            data: {
                message: '你好啊'
            }
        })
    </script>

v-cloak

v-bind

动态绑定属性

<div id="app">
        {{message}}
        <a v-bind:href="url">百度一下</a>
    </div>
    <script src="../js/vue.js"></script>
    <script>let app = new Vue({
            el: `#app`,
            data: {
                message: '你好啊',
                url:'https://www.baidu.com'
            }
        })
    </script>

语法糖（简写）：

<a :href="url">百度一下</a>

v-bind动态绑定class属性

点击按钮切换class

<!-- css -->
<style>
        .active{
            color: red;
        }
        .line{
            font-size: larger;
        }
</style>

<!-- html -->
    <div id="app">
        <h2 class="title" v-bind:class="{active:isActive,line:isLine}">{{message}}</h2>
        <button v-on:click="btnClick">切换</button>
    </div>
    <script src="../js/vue.js"></script>
    <script>let app = new Vue({
            el: `#app`,
            data: {
                message:'Hello Vuejs',
                isActive:true,
                isLine:true
            },
            methods:{
                btnClick:function(){
                    this.isActive=!this.isActive,
                    this.isLine=!this.isLine
                }
            }
        })
    </script>


绑定方式一、对象语法

用法有：

1、通过一个对象绑定一个类(可以传多个值)，并且和普通类同时存在并不冲突

<h2 class="title" v-bind:class="{active:isActive,line:isLine}">{{message}}</h2>

2、如果用法过于复杂，可以放在一个methods或者computed中

//hmtl
<h2 class="title" v-bind:class="getClasses()">{{message}}</h2>
//js
let app = new Vue({
            el: `#app`,
            data: {
                message:'Hello Vuejs',
                isActive:true,
                isLine:true
            },
            methods:{
                btnClick:function(){
                    this.isActive=!this.isActive,
                    this.isLine=!this.isLine
                },
                getClasses:function(){
                    return {active:this.isActive,line:this.isLine}
                }
            }
        })
绑定方式二、数组语法
<h2 class="title" :class="['active','line']">{{message}}</h2>

注意：其中active加引号就是字符串，不加引号就是变量

v-bind绑定style

对象语法

属性值需要加引号，使之解析成为字符串，不加引号就会解析成为变量,如下50px必须加引号，否则会被当成变量解析

对象的值可以来自于data中的属性

<h2 class="title" :style="{fontSize:'50px'}">{{message}}</h2>
<h2 class="title" :style="{fontSize:finaSize + 'px',backgroundColor:finaColor}">{{message}}</h2>

数组语法
<h2 class="title" :style="[baseStyle1 , baseStyle2]">{{message}}</h2>
data: {
     message:'Hello Vuejs',
     baseStyle1:{backgroundColor:'red'},
     baseStyle2:{fontSize:'100px'}
}
